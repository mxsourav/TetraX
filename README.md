# TetraX

One of the most compact bare minimum feathers cyber pen testing tool.

## Features
- **IR Jammer**: Includes a 38kHz channel for IR jamming.
- **TV Remotes**: Support for various TV remote brands (e.g., Samsung, LG, Sony, Panasonic).
- **AC Remote**: AC remote control functionality (*Note: Not stable yet*).
- **SD Card Support**: Integrated SD card support for extended storage and payloads.
- **SPA Page**: Comes with a Single-Page Application (SPA) interface for easy web-based control. We will be hosting this SPA for seamless access.
