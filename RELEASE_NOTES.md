# Release v1.0.0

## What's New in TetraX v1.0.0
- **IR Jammer**: Added 38kHz channel for IR jamming capabilities.
- **TV Remotes**: Support added for various brands (Samsung, LG, Sony, Panasonic, etc.).
- **AC Remote**: Initial AC remote functionality (*Note: Not stable yet*).
- **SD Card Support**: Integrated SD card support.
- **SPA Page**: Included our Single-Page Application (SPA) page. This will be hosted online after pushing.
